# Delivery Checklist

- Did the change cover all affected data types?
- Did field mapping type values remain consistent?
- Did backend and frontend both expose the capability?
- Did remote mode remain read-only in the UI?
- Did sync logic avoid hard delete by default?
- Did logs or batch records get updated where needed?
- Did backup naming and retention follow the fixed rule?
- Did downstream APIs stay read-only and Bearer-protected?
- Did validation run, or was the reason for skipping clearly stated?
